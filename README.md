# Legal Text Mining Project: ECtHR Dataset

## Project Overview
- AI/NLP project on English ECtHR judgments
- Dataset: ecthrnaacl2021 (JSONL)
- Tasks: Data exploration, preprocessing, topic modeling, outcome prediction

## Notebook Structure
- `01_data_exploration.ipynb`: Load & explore dataset
- `02_preprocessing.ipynb`: Clean & prepare text
- `03_modeling.ipynb`: Topic modeling
- `04_Trend & Topic.ipynb ` : Trend analysis over time using **yearly counts per topic
- `05_Citation & Precedent Network Analysis ` : Visualized top-cited or influential cases to understand legal authority propagation
- `results/`: Plots and analysis
- `src/`: Shared Python functions

## How to run
- Download dataset from Kaggle: [ecthrnaacl2021](https://www.kaggle.com/datasets/ecthrnaacl2021)
- Place dataset locally or use Kaggle notebook
- Open notebooks in Jupyter / Kaggle / Colab
